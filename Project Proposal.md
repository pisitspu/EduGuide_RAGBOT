Project Proposal

ชื่อโปรเจค: ครูแนะแนวออนไลน์ BY CHATBOT

Problem Statement

WHO ใครเดือดร้อน?: นักเรียนชั้น ม.ปลาย (ที่เตรียมสอบ TCAS) และผู้ปกครองที่กำลังวางแผนการศึกษาและประเมินค่าใช้จ่าย

WHAT ปัญหาคืออะไร?: ข้อมูลสำคัญ เช่น ค่าเทอมตลอดหลักสูตร วิทยาเขตที่ตั้ง และทุนการศึกษาของแต่ละมหาวิทยาลัย กระจัดกระจายอยู่ตามหน้าเว็บไซต์และไฟล์ PDF จำนวนมาก ทำให้ค้นหายาก เปรียบเทียบข้อมูลลำบาก และเสี่ยงต่อการได้รับข้อมูลที่ไม่อัปเดต

WHEN เกิดบ่อยแค่ไหน?: เกิดขึ้นเป็นประจำทุกปีในช่วงฤดูกาลเตรียมสอบเข้ามหาวิทยาลัย (TCAS) และทุกครั้งที่นักเรียนต้องการหาข้อมูลเพื่อประกอบการตัดสินใจเลือกคณะ

HOW MUCH เสียเวลา/เงินเท่าไหร่?: เสียเวลาค้นหาและรวบรวมข้อมูลด้วยตัวเองหลายชั่วโมงหรือหลายวันต่อคน และอาจสูญเสียโอกาสทางการศึกษาหากคำนวณค่าใช้จ่ายผิดพลาดหรือเข้าไม่ถึงข้อมูลทุนการศึกษา

Proposed Solution
สร้าง [ระบบ RAG Chatbot อัจฉริยะ] ที่ [รวบรวมและสืบค้นข้อมูลข้อเท็จจริง (Hard Data) เช่น ค่าเทอม และทุนการศึกษา จากฐานข้อมูลที่มีโครงสร้างเป็นระเบียบ (Structured Data)] เพื่อ [ช่วยให้นักเรียนและผู้ปกครองสามารถถาม-ตอบ และเข้าถึงข้อมูลที่ถูกต้องแม่นยำ เพื่อประกอบการตัดสินใจเลือกที่เรียนได้อย่างรวดเร็วในที่เดียว]

n8n Workflow Concept (สำหรับการออกแบบ Architecture)

Trigger: [Webhook] รับข้อความคำถามจากผู้ใช้งานผ่านช่องทางแชท

Process: [AI Agent / Vector Store Node] นำคำถามไปแปลงเป็น Vector และค้นหาข้อมูลบริบท (ค่าเทอม/ทุน) จากฐานข้อมูล (Retrieval) จากนั้นส่งให้ LLM สรุปคำตอบ

Action: เรียก [OpenAI API หรือ Gemini API] เพื่อประมวลผลข้อความ และเรียก [Pinecone / Qdrant API หรือ Vector DB ที่เลือกใช้] เพื่อดึงข้อมูล Master Data ที่เราทำ ETL ไว้

Notify: [LINE Messenger] ส่งคำตอบที่สรุปแล้วพร้อมลิงก์อ้างอิงกลับไปให้นักเรียนทาง LINE Official Account
